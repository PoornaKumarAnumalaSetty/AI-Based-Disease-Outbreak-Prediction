Welcome to the Disease Outbreak Predictor

This web application helps you predict potential disease outbreaks based on current case data. By entering details about your state, the disease, and the number of current cases, you can receive estimates of future case numbers.

Additionally, the application provides a bar graph comparing the predicted cases to the actual current cases, giving you a visual representation of the outbreak potential.

How It Works

The Disease Outbreak Predictor is an advanced web application that leverages historical data to forecast potential disease outbreaks. By allowing users to input current case numbers, disease types, and state information, the application generates accurate predictions along with visual representations of the data.

Key Features:

  • Utilizes a Random Forest Regression model trained on historical disease data to estimate future case counts.
  • Provides bar and pie charts to compare actual and predicted cases, facilitating better understanding of outbreak potential.

About This Project

The Disease Outbreak Predictor is a web application designed to forecast potential outbreaks of infectious diseases using machine learning. By analyzing historical data and current case numbers, it provides users with predicted future cases, helping in proactive disease management. Visualizations, such as bar graphs, enhance the understanding of predicted outcomes compared to actual cases.

Key Features:

  • Predictive Modeling: Utilizes Random Forest regression to estimate future disease cases.
  • User Input: Allows users to input state, disease, and current case numbers for accurate predictions.
  • Data Visualization: Generates bar graphs to compare actual and predicted cases, aiding decision-making.
  • Future Forecasting: Offers predictions for future outbreaks over specified timeframes.
  • Historical Data Analysis: Leverages historical disease data for informed decision-making and resource allocation.
